UAL default form elements, repsonsive & custom styles

Overview

Individual form controls automatically receive some global UAL styling. All textual <input>, <textarea>, and <select> elements are set to width: 100%; by default.

UAL styled select boxes must follow a specific html structure.
Please refer to the UAL select box guidelines below for more information.

Use the class .with-box-shadow on the main <form> tag to give forms a subtle drop shadow.

Unstyled / Default elements

Used in cases where the HTML for checkboxes must have the label before the checkbox (eg Salesforce for events). Checkboxes appear unstyled if given the 'unstyled' class. The label for the checkbox should also be given a class of 'unstyled'.
