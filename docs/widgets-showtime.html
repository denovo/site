Widgets: Showtime Feed test for update

Example: Latest 3 showtime images from CSM, LCF & Wimbledon, sorted by date

Requires JS file: ual-get-showtime.js to be loaded into the page and then initialized using the code below.

HTML code: 
        
        <div id="showtime-feed" >
          <img src="http://artslondon.github.io/beta/assets/img/loader.gif" alt="loading..."/>
        </div>
Javascript: 
        
        <script> 
        window.onload=function(){ 
          // load in showtime images 
          fetchShowtimeGallery(3, ['csm','lcf','wimbledon'], 'thumb', 'screen', 'showtime-feed'); 
        } 
        </script>
