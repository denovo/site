Example Properties & Usage
UAL Black color: 000000 (used with text more than 16px)
Soft Black color: 282828 (used with text less than or equal to 16px)
AAA gray color: 929292 (used with white background only. passes AA rating for text 14px and AAA rating for text 18px and up)
Light Gray color: C8C8C8 (used for visual elements only. not for use with overlayed text)
Background Gray color: EEEEEE (must be used with text set in ual black/000000 to pass AAA rating for text colour contrast)
