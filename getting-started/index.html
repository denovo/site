A responsive, proportional, nestable, and mobile-first 16-column grid framework.

The grid system uses gridset by Mark Boulton as its foundation. Gridset is a tool for making grids, and you can find out more by visiting their document page.

The grid system has three layouts, Desktop, Tablet and mobile

Mobile
- 320px - 599px
Tablet
- 600px - 959px
Desktop
- 960px - 1280px
